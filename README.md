## 📌 Project Overview
The **Diabetes Prediction System** is a Machine Learning project developed using **Python**.  
It predicts whether a person is likely to have **diabetes** based on medical attributes such as glucose level, BMI, age, and other health-related features.

This project uses the **Pima Indians Diabetes Dataset**, which is widely used for healthcare machine learning research.

---

## 🎯 Project Objectives
✔ Analyze medical data  
✔ Train a Machine Learning model  
✔ Predict diabetes outcomes  
✔ Visualize prediction results  

---

## 🗂 Dataset Information

The dataset contains several medical predictor variables and one target variable (**Outcome**).

| Feature | Description |
|-------|-------------|
| Pregnancies | Number of pregnancies |
| Glucose | Plasma glucose concentration |
| BloodPressure | Diastolic blood pressure |
| SkinThickness | Triceps skin fold thickness |
| Insulin | 2-Hour serum insulin |
| BMI | Body Mass Index |
| DiabetesPedigreeFunction | Diabetes family history function |
| Age | Age of patient |
| Outcome | 0 = Non-Diabetic, 1 = Diabetic |
